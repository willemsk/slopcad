## 🤖 What is this? (Vibe Coding Exercise)
**Please note**: This repository is primarily an exercise and learning experience in **"vibe coding"**—using AI agents and LLMs to iteratively generate, architect, and refine an entire software application from scratch. 

While SlopCAD is a genuinely functional, highly-responsive browser-based CAD tool with drawing features, snapping, and constraint solving, it is **not** meant to be taken too seriously or used for critical architectural production. It serves as a fun, open-source experiment to demonstrate how far AI-assisted development can go in building complex, interactive HTML5 Canvas applications!

## 🚀 Quick Start
If you just want to spin up the local development environment and play around with the canvas:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/willemsk/slopcad.git
   cd slopcad
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the Vite dev server:**
   ```bash
   npm run dev
   ```
4. **Open your browser** to the local URL (usually `http://localhost:5173`).
