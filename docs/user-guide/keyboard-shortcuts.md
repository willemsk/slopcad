# Keyboard Shortcuts

Accelerate your workflow with these keyboard shortcuts in SlopCAD.

---

## 🛠️ Tool Selection

| Shortcut | Action |
|----------|--------|
| `S` | Select Tool |
| `W` | Wall Tool |
| `D` | Door Tool |
| `N` | Window Tool |
| `L` | Line Tool |
| `R` | Rectangle Tool |
| `C` | Circle Tool |
| `A` | Arc Tool |
| `T` | Text Tool |
| `M` | Dimension Tool |

---

## ⌨️ Command & View Operations

| Shortcut | Action |
|----------|--------|
| `Escape` | Cancel active tool or clear selection |
| `Delete` / `Backspace` | Delete selected entities |
| `Ctrl + Z` | Undo last operation |
| `Ctrl + Y` | Redo undone operation |
| `Ctrl + S` | Save project locally |
| `Space` / `Enter` | Confirm command line input |

---

## 🖱️ Mouse Navigation

- **Pan**: Hold Middle Mouse Button (scroll wheel) and drag, or hold `Space` and drag with left click.
- **Zoom**: Scroll up to zoom in, scroll down to zoom out (zooms to cursor location).
- **Selection**: Click an entity to select it. Click empty canvas and drag to make a window/crossing selection.
